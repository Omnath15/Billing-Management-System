# Billing Management System (Python)

## Description
This project is a Billing Management System developed using Python and MySQL to automate the process of generating and managing customer bills efficiently.

## Features
- Customer details entry
- Item-wise billing
- Automatic total calculation
- Invoice generation
- Data storage using MySQL
- Export data to Excel

## Technologies Used
- Python
- Tkinter
- MySQL
- SQL

## How to Run
1. Install Python
2. Setup MySQL database
3. Run the main Python file

## Purpose
To simplify billing operations and maintain accurate records digitally.
